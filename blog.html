What is Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, allowing scripts to access, modify, and manipulate the HTML or XML content dynamically.

How do you select an element from the DOM?

1.getElementById() selects by ID

2.getElementsByClassName() selects by class name.

3.getElementsByTagName() selects by tag name.

4.querySelector() selects the first element matching a CSS selector.

5.querySelectorAll() selects all elements matching a CSS selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to manage events on its child elements. Instead of adding event listeners to multiple child elements, the parent listens for events, leveraging event bubbling. It's useful because it improves performance (fewer listeners) and simplifies code, especially when dynamically adding or removing child elements.

How do you manipulate an element's attribute and styles using the DOM?

To manipulate an element's attributes and styles in the DOM: 1. Attributes & 2. Styles

Attributes: element.setAttribute('attribute', 'value') Sets or updates an attribute. element.getAttribute('attribute') Retrieves the attribute value.element.removeAttribute('attribute') Removes an attribute.


Styles: element.style.property = 'value' Directly changes the inline CSS style (e.g., element.style.color = 'red'). Alternatively, use element.classList.add(), remove(), or toggle() to manage classes for styling.
